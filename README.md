# ReadyCBAHI System - V3 Production

## 🇸🇦 نظام متابعة استدامة جودة الأداء والامتثال - تجمع الجوف الصحي

نظام متكامل ومقدم احترافياً لإدارة ومتابعة جودة الأداء والامتثال للمعايير الصحية (CBAHI) داخل منشآت تجمع الجوف الصحي. يتميز النظام بواجهة عصرية تعتمد على مفهوم الـ **Glassmorphism** وتجربة مستخدم سلسة جداً.

---

## ✨ المميزات الرئيسية (Key Features)

- **💎 واجهة Glassmorphism:** تصميم عصري، شفاف، وجذاب يوفر تجربة مستخدم متميزة.
- **📊 تحليل إحصائي متقدم:** شاشات تفاعلية تعرض نسب الامتثال والنتائج بالرسوم البيانية.
- **📁 تصدير البيانات:** دعم كامل لتصدير التقارير بصيغة **Excel** و **PDF**.
- **⚖️ نظام المقارنات:** مقارنة ذكية بين أداء عدة مراكز أو منشآت طبية في آن واحد.
- **🎬 عرض تقديمي مدمج:** عرض تقديمي داخلي (Presentation) مدعوم بالصوت (Speech Synthesis) يشرح خصائص النظام.
- **📱 توافق كامل مع الجوال:** واجهة متجاوبة (Responsive) تعمل بكفاءة على الهواتف الذكية والأجهزة اللوحية.
- **✨ حركات انتقالية سلسة:** استخدام **Framer Motion** لإضافة لمسات جمالية عند التنقل بين الشاشات.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Frontend:** React.js, Vite, Framer Motion, Lucide React, XLSX.
- **Backend:** Node.js, Express.
- **Database:** PostgreSQL / SQLite (حسب الإعداد).
- **Styling:** Vanilla CSS (Glassmorphism design system).

---

## 🚀 كيفية التشغيل (Getting Started)

### 1. المتطلبات (Prerequisites)
- [Node.js](https://nodejs.org/) الإصدار الأحدث.

### 2. التثبيت (Installation)
قم بتثبيت الملحقات في المجلدين الرئيسي والفرعي:
```bash
# في المجلد الرئيسي (Root)
npm install

# في مجلد الفرونت إند
cd frontend
npm install
```

### 3. التشغيل (Usage)
تشغيل الباك إند (Backend):
```bash
cd backend
npm start
```

تشغيل الفرونت إند (Frontend):
```bash
cd frontend
npm run dev
```
